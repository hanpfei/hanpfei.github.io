中文技术网站排名

Posted on 2018-01-13 | In 业界趣闻 | | Visitors:

中文技术网站对整个华语圈的技术工作者及技术学习者的工作和学习提供了巨大的帮助。

我们常用的中文技术网站的排名如下表（排名数据通过以域名 alexa.chinaz 查询获得）：

CSDN 不愧为中国最大的IT社区和服务平台，在各项排名中都名列第一。阮一峰的网络日志 和 酷壳 分别是阮一峰老师和陈皓老师的博客，但在访问量和排名上，甚至比许多专门的技术网站还要考前。简书 本身不是一个纯粹的技术类网站，但有众多的开发者通过这一平台来分享及学习技术相关的知识，因而也一并列在这里。ITeye 主要是 Java 的开发交流社区，ImportNew 专注于 Java 和 Android 开发，其它网站则多是综合性技术网站。
