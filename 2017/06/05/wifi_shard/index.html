  1. 共享网络连接给共享 WiFi AP
    点击开始， 选择 控制面板 打开控制面板。点击 网络和 Internet -> 网络和共享中心 -> 网络和共享中心 。可以看到一块 Microsoft Virtual WiFi Miniport Adapter 虚拟网卡：

    选中正在联网的网卡，如上图中的 无线网络连接，右键单击，选择 属性 -> 共享：

    选中 允许共享，并选择虚拟网卡 无线网络连接3。如上图所示。
    至此，无线网络共享打开成功。不过用这种方法设置 Win7 热点会出现一些问题，比如说每次开机都要设置一次（当然也可以批量处理解决这个问题）。

系统要求：带有线和无线网卡的笔记本电脑。
